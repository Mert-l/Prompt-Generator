# Prompt-Generator
Collecting TikTok hashtag data from Apify, processing it, and generating AI-powered viral video prompts

## Overview
The script fetches trending TikTok hashtag data via **Apify**, processes insights such as audience demographics, interests, and related hashtags, and then uses **Groq AI** to generate short, attention-grabbing video prompts optimized for TikTok. The prompts include suggested captions and hashtags, making it easy to directly create engaging content.

## Requirements
- Python 3.9+
- [Apify API Key](https://apify.com/)
- [Groq API Key](https://www.groq.com/)
- Packages: `apify-client`, `groq`
